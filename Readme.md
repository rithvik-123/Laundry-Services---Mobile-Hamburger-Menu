# Task 8: CSS Hamburger Menu in Mobile (Pseudo Classes)

**Author:** D P Rithvik Kumar

## How to Run
1. Unzip the folder and keep `index.html` and `style.css` together.
2. Open `index.html` in your browser.
3. Shrink the screen width to below 768px to reveal the hamburger menu icon.
4. Click the hamburger icon to see the CSS-only X animation and the slide-in menu.
5. Click the "Close X" button inside the menu to see how it steals focus and dismisses the menu!
